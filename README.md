# Thread Synchronization Visual Demonstrator

## Overview
A Java Swing-based application that visually demonstrates classic thread synchronization problems.

## Problems Covered
- Producer–Consumer
- Dining Philosophers
- Readers–Writers

## Synchronization Methods
- Semaphores
- Monitors (Locks & Conditions)
- Unsafe (No Synchronization)

## Technologies Used
- Java
- Swing
- Multithreading
- java.util.concurrent

## How to Run
1. Compile `ThreadSyncDemo.java`
2. Run the main class
3. Select a problem and synchronization method
4. Click Start

## Learning Outcomes
- Race conditions
- Deadlock prevention
- Thread communication
- UI thread safety
